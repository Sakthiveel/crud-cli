Custom CLI Tool For CRUD Operations:


Installation:

Install the dependencies
$ npm install

Create Symlink
$ npm link


Commands
List users (list or l)
$ crud-cli list

Find Customers (find or f)
$ crud-cli find [NAME]


Add Customer (add or a)
$ crud-cli add

Update Customer (update or u)
$ crud-cli update [_ID]

Remove Customer (remove or r)
$ crud-cli remove [_ID]
